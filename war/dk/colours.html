Brand Colour Palette

The colour swatches below display the colour name, examples of where and how they should be used as well as a hex number for use in the creation of new components.

Reflection Purple

SASS Variable:
$reflection-purple
  • Used sparingly as primary accent colour
  • Active clickable links and major buttons
  • Hero area backgrounds
  • Selected states in Navigation panels

Reflection Red

SASS Variable:
$reflection-red
  • Used as a hover colour on major calls to action to emphasise importance

Reflection Green

SASS Variable:
$reflection-green
  • Used as a selected state and a positive indicator of an action having been performed successfully

Dark Grey 1

SASS Variable:
$dark-grey-1
  • Main heading colour
  • Slide out panel backgrounds
  • Bolded main body text

Dark Grey 2

SASS Variable:
$dark-grey-2
  • Tab titles
  • Top bar background
  • Content labels

Mid Grey 1

SASS Variable:
$mid-grey-1
  • Main body copy
  • Form field text hover

Mid Grey 2

SASS Variable:
$mid-grey-2
  • Text on a dark background
  • Icons on a dark background

Light Grey 1

SASS Variable:
$light-grey-1
  • Text on a dark background
  • Hover states on a dark background

Light Grey 2

SASS Variable:
$light-grey-2
  • Light grey text labels
  • Selected icons on dark backgrounds

Reflection White

SASS Variable:
$reflection-white
  • Main content area background
  • Selected text, icons and button text

Panel Grey

SASS Variable:
$panel-grey
  • Sitewide content panel backgrounds

Graph Fill Grey

SASS Variable:
$graph-fill-grey
  • Graph fill area background
  • Light grey text

Divider Grey

SASS Variable:
$divider-grey
  • Content dividers
  • Light grey text
