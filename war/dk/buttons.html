Primary Button Small

Small purple buttons are used for primary calls to action which require prominence but where space is limited (for example the ‘Apply’ button at in the top menu bar).

There are two types of padding as shown on the right: 20px and 30px on either side of the internal text label. The 30px padding is used for buttons with short text labels to ensure the button is still large enough to press.

Disabled

Active


									// Disabled
									<button class="ref-button--cta-small" disabled>Lorem</button>
									<button class="ref-button--cta-small--wide" disabled>Lorem</button>

									// Active
									<button class="ref-button--cta-small">Lorem</button>
									<button class="ref-button--cta-small--wide">Lorem</button>

Fluid Width Button

Fluid width buttons are used in forms and dialogue boxes where the button width is determined by the width of the preceeding form elements or the button’s container.
