Helper Tooltip

These pop ups appear when a user hovers on an object (such as a filter tool) after a short delay and contain a brief description of the objects function. They act as a prompt to help the user identify certain important elements or features on a page.

The tooltip always appears above the item that the user interacts with. (see examples). They should contain as few words as possible, they serve as a prompt rather than an explanation.

Lightbox

When an action needs to be performed or a decision needs to be confirmed without taking the user away from the page they are working on a lightbox is used. This is to ensure a smooth user journey, allowing them to return to what they were doing once the action is performed.

The box appears as a lightbox centred over the existing content and can be closed simply by clicking away from the box itself.

If required the box can contain several steps for the user to move through.

Alert Toast

Alert toasts pop up from the bottom right corner of the screen every time a notification is sent to the user. They contain a summary title and when clicked link to the notifications page where the full alert can be read and acted upon if necessary. They appear for a few seconds and then disappear unless they require urgent attention in which case they remain in position until clicked or closed.

Alert toasts appear in conjunction with a small alert bubble in the 'My Account' button in the site header. The bubble indicates how many unread alerts the user has and remains visible until there are no more unread notifications.
