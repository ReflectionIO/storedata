Structure and classes

Wrap tabs in a container with class "tabs-container". Each tab requires the class "tabs__tab" and set the default tab to also have class "is-active".

The <a> tag should have an href of the id of the container of the content the tab link opens (href="#tab-content-one", for instance). The <a> tag should also have the classes "tabs__link" and "js-tab-select" for styling and JavaScript functionality.

The container of the content that the tabs link to should be within an element with class "tabs__content-container". This should contain a list of items with class "tabs__content", the default content also having the class "tabs__content--is-showing". These items should have the id's that the tabs link to (id="tab-content-one", for example).

// Tabs structure
<ul class="tabs-container">
 <li class="grid__column grid__column--one-quarter tabs__tab is-active js-tab-select">
  <div class="tab-animate-active"></div>
  <a href="#tab-content-one" class="tabs__link">
   <span class="ref-icon-before ref-icon-before--cog"><span>Revenue</span></span>
  <a>
 </li>
 ... etc
</ul>
